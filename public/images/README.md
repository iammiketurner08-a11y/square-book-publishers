# Website Images

You can customize any image on the website by replacing the files inside this folder.
To keep the website looking great, use the exact same filenames and formats (or update the file paths in the source files inside `/src/pages/`).

## Image Reference Guide:

| Page | Filename | Description | Recommended Dimensions |
|------|----------|-------------|------------------------|
| **Home Page / Hero** | `hero.svg` | Main illustration/photo for the landing page hero section. | `1200 x 1600` or `3:4` |
| **Home Page / Process** | `process.svg` | Illustration showcasing the manuscript-to-marketplace flow. | `800 x 800` or `1:1` |
| **Home Page / Team** | `team.svg` | Section background or photo of our expert publishing team. | `800 x 600` or `4:3` |
| **Services Page / Consultation** | `services-consult.svg` | Heading image for publishing consult services. | `800 x 600` or `4:3` |
| **About Page / Mission** | `about-mission.svg` | Illustration for our publishing mission & core values. | `800 x 1000` or `4:5` |
| **Process Page** | `process-step-0.svg`<br>`process-step-1.svg`<br>`process-step-2.svg`<br>`process-step-3.svg` | Beautiful custom graphics representing each step of publication. | `800 x 600` or `4:3` |
| **Portfolio / Book Covers** | `book1.svg` (The Silent Horizon)<br>`book2.svg` (Digital Renaissance)<br>`book3.svg` (The Little Dreamer)<br>`book4.svg` (Beyond the Veil)<br>`book5.svg` (Modern Leadership)<br>`book6.svg` (Echoes of Time) | Hardcover/softcover book covers showcased in the portfolio. | `600 x 800` or `3:4` |
| **Testimonials / Avatars** | `author1.svg`<br>`author2.svg`<br>`author3.svg`<br>`author4.svg` | User avatars in the client and author testomonials loop. | `100 x 100` or `1:1` |

---

*Note: All images are initially rendered using modern, high-contrast vector illustrative graphics (`.svg`) for maximum sharpness and absolute zero external network dependencies. You can drop in any `.png`, `.jpg`, `.webp`, or `.svg` files to replace them.*
